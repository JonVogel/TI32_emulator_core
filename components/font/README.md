# TI_Font

8×8 bitmap font matching the TI-99/4A's character ROM, plus the Texas Instruments logo glyphs used on the boot screen. Header-only Arduino library.

## Used by

- esp32-s3-box-basic
- ti-extended-basic-esp32
- ti-basic-otg
- ti-scott-adams-esp32

## Embedding

```sh
git submodule add https://github.com/JonVogel/TI_Font.git
```
